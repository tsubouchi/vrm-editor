# VRM Editor

VRMモデルを編集・カスタマイズするためのウェブアプリケーション。

## 機能

- VRMモデルのアップロードと表示
- モデルの表情、ポーズの調整
- テクスチャの編集
- モデルのエクスポート

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install
# または
yarn install
# または
pnpm install

# 開発サーバーの起動
npm run dev
# または
yarn dev
# または
pnpm dev
```

## 使用技術

- Next.js 15
- Three.js / React Three Fiber
- @pixiv/three-vrm
- Tailwind CSS
- Radix UI
